# 100-days-of-soc-journey


Today I learned about SOC (Security Operations Center).

SOC is a team that monitors systems 24/7 to detect, investigate, and respond to security threats.

I also learned what a SOC team does:

- Monitor alerts from different security tools  
- Detect suspicious activity  
- Analyze incidents and respond properly  

I understood why SOC matters:

- It helps protect organizations from attacks  
- It keeps sensitive data secure  
- It reduces risk by detecting threats early  

This is just the beginning. Learning step by step and staying consistent.
